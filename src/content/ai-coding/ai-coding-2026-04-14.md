      Model Context Protocol adoption is accelerating across ALL major tools. Codex CLI added MCP tool declarations with structured output schemas, Copilot CLI launched MCP server installation from a registry, and Claude Code expanded MCP result-size limits to 500K chars. MCP is becoming the universal plugin interface for AI coding tools.
      Autonomous agents mature &nbsp;
      The shift from "suggest code" to "execute entire workflows" is now table stakes. Copilot's cloud agent handles merge conflicts and turns issues into PRs. Claude Code can now drive native GUIs via computer use. Codex's Realtime V2 enables background agent progress streaming. Every vendor is racing toward fully autonomous coding agents that can plan, execute, and verify.
      BYOK &amp; model flexibility &nbsp;
      Copilot now supports BYOK with OpenRouter, Ollama, and other providers. Codex launched pay-as-you-go Codex-only seats. The walled-garden model is cracking — developers want to choose their own models and providers, and vendors are adapting.

      CLI-based coding assistants are surging: Codex CLI hit 3M weekly users (5x in 3 months), Copilot CLI got its own usage metrics, and Claude Code pushed 30+ releases in 5 weeks. The terminal-first workflow is winning developer mindshare alongside traditional IDE extensions.
