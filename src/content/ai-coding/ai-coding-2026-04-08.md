## &#x1F50D; Notable Trends
    **Plugin ecosystems are maturing across the board.** Both Claude Code (plugin executables, frontmatter-based skills) and OpenAI Codex (first-class plugin sync, `/plugins` browser) shipped significant plugin infrastructure this week — a clear sign that extensibility is the next battleground after base agent capabilities. GitHub Copilot joined with the **Copilot SDK** in public preview, giving developers programmatic access to build custom integrations.
    **Bring Your Own Model (BYOM) is becoming standard.** Copilot CLI now supports BYOK and local models. Claude Code added Bedrock/Mantle support with an interactive setup wizard. This flexibility signals the shift from vendor lock-in to platform-agnostic agent infrastructure.
    **Enterprise governance is tightening.** Copilot's signed commits, org-level firewall controls, and custom instructions GA all point toward cloud agents earning trust in regulated environments. Claude Code's managed settings enforcement (`forceRemoteSettingsRefresh`) follows the same trajectory.
    **Model churn continues at OpenAI.** GPT-5.3-Codex is now the default, older models are being aggressively deprecated (removal on Apr 14), and GPT-5.4/5.4-mini are the current frontier. Gemini is also pushing forward with 3.1 Pro and 3.0 Flash in preview.
