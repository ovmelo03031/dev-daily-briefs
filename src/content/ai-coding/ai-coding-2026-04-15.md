## Notable Trends
    **Agent autonomy is the new battleground.** GitHub Copilot's Autopilot, Claude Code's monitor tool + plugin background monitors, and OpenAI Codex's streaming agent progress all point to the same thing: coding assistants are racing toward fully autonomous multi-step workflows where the developer steers rather than types. Copilot's Remote CLI sessions &mdash; controlling agents from your phone &mdash; takes this further than anyone else this month.
    **Chinese labs are punching at frontier level.** Zhipu's GLM-5.1 claiming #1 on SWE-Bench Pro (58.4) above GPT-5.4 and Claude Opus 4.6 is the headline, but Alibaba's Qwen3.6-Plus and Moonshot's Kimi Code K2.6 show the ecosystem is broad and accelerating. The open-weight strategy from these labs (GLM-5.1 fully open-source, Qwen3.5-Omni open) continues to pressure Western proprietary models.
    **Security-specialized models emerge.** Both Anthropic (Claude Mythos under Project Glasswing) and OpenAI (GPT-5.4-Cyber with tiered access) launched restricted-access models targeting cybersecurity &mdash; a signal that frontier capabilities are splitting into domain-specific tracks with controlled distribution. Meanwhile, Claude Code's subprocess sandboxing and PID namespace isolation show security hardening happening at the tool layer too.
    **MCP is becoming infrastructure.** Copilot CLI v1.0.25 adds MCP server installation from a registry with guided setup, Claude Code continues deepening plugin/MCP support, and Google's Code Assist adds customization via agent mode &mdash; the Model Context Protocol is quietly becoming the integration standard for AI coding tools.
