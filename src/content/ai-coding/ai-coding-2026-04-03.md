## 📊 Notable Trends
    **Agentic SDKs are proliferating.** GitHub shipped the Copilot SDK (Node, Python, Go, .NET, Java), OpenAI Codex landed a multi-agent v2 protocol with structured inter-agent messaging, and Claude Code added plugin executables — all moving toward "embed the agent in YOUR infra" rather than just IDE extensions. The CLI-as-platform era is here.
    **Persistent memory is becoming table stakes.** Gemini Code Assist on GitHub now stores cross-session context, and Copilot Memory is enabled by default for Pro users. Claude Code's hook system (defer/resume, PermissionDenied retry) pushes in the same direction — tools that remember context and recover gracefully from interruptions.
    **Model churn continues at pace.** OpenAI retired GPT-4o/4.1/o4-mini and pushed GPT-5.3-Codex as LTS while shipping GPT-5.4 GA. Google brought Gemini 3.1 Pro to Code Assist. The models underneath coding tools are cycling faster than ever — watch for deprecation notices in your CI pipelines.
    **Privacy under pressure.** GitHub's new policy to use Copilot Free/Pro/Pro+ interaction data for model training (effective April 24, opt-out required) is a significant shift. Organizations on Business/Enterprise plans are exempt, but individual developers should review their settings.
