<h2>Notable Trends</h2>

<p><strong>Chinese open-weight models are challenging frontier proprietary ones head-on.</strong> Zhipu's GLM-5.1 now leads SWE-Bench Pro (58.4) ahead of GPT-5.4 and Claude Opus 4.6, while Alibaba's Qwen 3.6 Plus tops Terminal-Bench 2.0 with a 1M context window — both under permissive open licenses. The cost gap is staggering: GLM-5 API pricing runs 5-8x cheaper than Opus 4.6, and Doubao 2.0 claims 10x cost reductions. The open-source coding advantage from China is real and accelerating.</p>

<p><strong>Agent mode is now table stakes for every coding tool.</strong> Claude Code expanded Remote Control to mobile clients, Copilot launched fully autonomous "Autopilot" sessions with browser debugging, Gemini Code Assist added agent mode customization, and Codex CLI introduced remote workflows with websocket transport. The new frontier isn't "can the AI write code?" — it's "can the AI run entire development workflows unsupervised?"</p>

<p><strong>Cross-device and cross-platform control is the new competitive axis.</strong> Both Claude Code (push notifications + mobile remote control) and GitHub Copilot (remote CLI sessions from web/mobile) are racing to let developers steer AI agents from anywhere. The coding assistant is becoming a background service you monitor, not a tool you sit in front of.</p>

<p><strong>DeepSeek V4 remains the most anticipated unreleased model.</strong> Announced in mid-February with a 1-trillion-parameter MoE architecture, it still hasn't shipped as of April 16. Multiple sources keep saying "within weeks," making it the Godot of the AI model world. Meanwhile, competitors aren't waiting.</p>
