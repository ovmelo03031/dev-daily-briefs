## 📡 Notable Trends
    **Hooks are everywhere.** Claude Code, Codex CLI, and GitHub Copilot all shipped new hook systems this month — enabling custom automation at key agent lifecycle events. This signals a shift from "AI writes code" to "AI plugs into your existing workflows."
    **MCP as the universal connector.** Claude Code's MCP elicitation, channels research preview, and Copilot's MCP auto-approve support show the Model Context Protocol becoming the standard interface between AI agents and external tools. Codex's plugin setup improvements follow the same trajectory.
    **Intent-based editing gains ground.** Gemini's "Finish Changes" is the most explicit example — no prompt needed, just start editing and the AI infers the rest. Combined with Claude Code's effort frontmatter and Copilot's plan agent going GA, the tools are betting that developers prefer showing intent over writing explicit instructions.
    **Model consolidation continues.** OpenAI retired GPT-4o and o4-mini from ChatGPT while GitHub restricted premium model access for students. The industry is narrowing to fewer, more capable reasoning models (o3, Opus 4.6, Gemini 3.0) rather than maintaining broad model catalogs.
