<div class="trends">
<h2>&#x1F4CA; Tendencias Destacadas</h2>
<p>Anthropic dominó esta semana con Claude Opus 4.7 recuperando el liderazgo en SWE-bench Pro (64.3%) apenas seis días después del lanzamiento de Opus 4.6 — un ritmo de iteración sin precedentes. Mientras tanto, cada herramienta de coding ahora incluye computer use o navegador integrado: Codex añadió ambos el mismo día, Copilot ganó resolución de conflictos de merge vía cloud agents, y "Finish Changes" de Gemini Code Assist llegó a GA.</p>
<p>El ecosistema de labs chinos sigue acelerando: GLM-5.1 de Z.ai lideró silenciosamente SWE-Bench Pro con 58.4%, Moonshot AI confirmó Kimi K2.6 Code Preview en beta, y Qwen 3.6 Plus de Alibaba ya lidera en varios benchmarks. El giro estratégico de Meta del open-source Llama al propietario Muse Spark marca un cambio filosófico mayor, mientras DeepSeek V4 sigue siendo el modelo no lanzado más anticipado, ahora esperado en semanas sobre hardware Huawei.</p>
<p>La seguridad emerge como diferenciador: OpenAI lanzó GPT-5.4-Cyber para profesionales verificados con ingeniería inversa de binarios, Copilot añadió residencia de datos US/EU con FedRAMP Moderate, y Claude Opus 4.7 incluye detección y bloqueo automatizado para usos prohibidos de ciberseguridad. La era del tooling "security-first" está llegando.</p>
